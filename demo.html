1. 安装 git 2. 打开 git bash 黑色的窗口 3. 注册 github的账号(邮箱验证一下) 4. 登陆 github 5. 创建一个新的仓库 点击右上角的+ 6. 给仓库起一个名字 7. 在你的电脑上的任何盘符下面创建一个文件夹(英文命名) 8. 在phpStorm里面打开这个文件夹 9. 在这个文件夹下面创建一个任意的文件 10. copy path 拷贝这个文件夹的路径 11.来到黑色的窗口，输入 cd之后 将这个文件夹的路径粘贴到 git bash（黑色的窗口） 12. 来到这个文件夹 13. git init 初始化 文件夹 14. git add . 15. git commit -m "xxxxx" 16. 将github的地址告诉本地的git （拷贝github仓库的地址：git remote add origin "仓库的地址"），运性 17. git push origin master 18. 等待弹出一个窗口，让你输入你的github的用户名和密码 19. 文件等待上传 20. 上传成功 hello world
